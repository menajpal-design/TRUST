# 🇧🇩 TRUST - UnionDesk Enterprise SaaS Organization ERP Platform

**UnionDesk (TRUST)** is a production-ready, Enterprise SaaS Organization Management System built specifically for administrative governance, 13-tier committee hierarchies, Bangladesh 9-tier geographic administrative units, financial cashbook ledgers, digital QR tickets & receipts, member directory, and AI-powered executive synthesis.

---

## 🌟 Key Features

- **🇧🇩 Bangladesh 9-Tier Geographic Management System**: Preloaded Division (8), District (64), Upazila (495+), Municipality/City Corp, Union, Ward, Village, Mohalla, and Custom Area hierarchy with cascading selectors, 1-click sync, and Excel import/export.
- **🏛️ 13-Tier Bangladesh Committee Hierarchy**: National, Division, District, Upazila, Union, Ward, Village, School, College, Mosque, Market, Women's, and Youth committees.
- **⚙️ 17-Category Enterprise Settings Hub**: General, Organization, Members, Finance, Committees, Attendance, Notifications, Security, Appearance, Permissions, Backup, Subscription, AI, API, and System policies.
- **💼 Double-Entry Cashbook Ledger & Financial Governance**: Daily cashbook, period closings, income/expense recording, fiscal budgets, and automated QR payment receipts.
- **🎟️ Events & Digital QR Tickets**: RSVP, ticket generation, QR ticket verification, and geofenced attendance.
- **✨ Gemini AI ERP Copilot**: AI financial insights, notice generator, meeting summarizer, and chat assistant.
- **🚀 Vercel Ready**: Full serverless deployment configuration for Vercel.

---

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, TailwindCSS, Zustand, Axios, Lucide React, Socket.io-client.
- **Backend**: Node.js, Express, MongoDB (Mongoose), JSON Web Tokens (JWT), Winston, Zod, ExcelJS, PDFKit, QRCode, Socket.io.
- **Deployment**: Vercel Serverless Functions (`@vercel/node` & `@vercel/vite`).

## 🚀 Getting Started

### 1. Installation
```bash
# Clone the repository
git clone https://github.com/menajpal-design/TRUST.git
cd TRUST

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### 2. Development Mode
```bash
# Start backend server
cd server
npm run dev

# Start frontend client
cd client
npm run dev
```
